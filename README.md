### Requirements
- docker
- docker-compose

### To run project:

- run docker-compose
```bash
docker-compose up -d
```

- install dependencies
```bash
sail composer install
```

- Install dependencies
```bash
sail composer install
```

- Generate app key
```bash
sail artisan key:generate
```

- Run migrations
```bash
sail artisan migrate
```

- Run seeds
```bash
sail artisan db:seed
```

### Use in browser: `localhost`
